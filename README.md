# DIY 12Nm Direct-Drive Racing Wheel & Load-Cell Pedals

A high-performance, custom-built sim racing setup engineered from scratch using a high-torque hoverboard motor.

## 🚀 Project Overview
The goal of this project is to build an  racing simulation setup that replicates the performance of high-end commercial direct-drive systems at a fraction of the cost. 

- **Wheelbase:** Powered by a hoverboard motor running Odrive-Wheel via an ODESC controller to deliver up to 12Nm of realistic force feedback .
- **Pedals:** A multi-pedal system featuring a accurat hydraulic like feel brake driven by a load-cell sensor, alongside Hall effect sensors for gas and clutch inputs.
- **Structure:** Custom struction made from CNC pre macined parts and PLA+ parts for the most amount of rigidity
